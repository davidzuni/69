GIGABYTE:

Múltiplo de Byte, unidad de almacenamiento de información. Se utiliza para dispositivos de gran capacidad, como discos duros y DVD. Véase Byte.
GIF:

Formato de imágenes y animaciones muy utilizado para transmitir imágenes en internet. Las siglas provienen de Graphics Interchange Format. Optimiza el tamaño de los dibujos (no tanto de las fotografías, para las que es más conveniente el formato .jpg). Admite la transparencia de un color y permite la construcción de animaciones.
GUSANO:

Es un tipo de virus. Se trata de un programa que se copia a sí mismo hasta ocupar toda la memoria del disco. Para eliminarlo es necesario utilizar un programa antivirus.
